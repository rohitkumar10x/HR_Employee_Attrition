# HR Employee Attrition Analysis (Power BI Project)

This project analyzes employee attrition to understand **why employees leave the company** and what factors drive turnover.  
Using Power BI, the dashboard visualizes trends based on **satisfaction, salary, promotions, workload, tenure, and departments**, helping HR make data-driven decisions to reduce attrition.

## 📊 Project Objective

These metrics reveal a **high attrition rate**, which directly affects productivity and hiring costs.

---

## 📉 Dashboard Insights (Important)

### 1. Salary Level  
Low-salary employees show the **highest attrition**, indicating compensation dissatisfaction.

### 2. Promotion Impact  
Employees without a promotion in the last 5 years leave more frequently.

### 3. Department Risk  
Sales, Technical, and Support departments have the **highest attrition**, showing heavy workload or pressure.

### 4. Tenure Trend  
Employees with **4–5 years** of service leave the most, suggesting mid-career dissatisfaction.

### 5. Satisfaction Levels  
Stayed employees have **higher satisfaction** compared to those who left.

### 6. Workload Pattern  
Employees who left show **imbalanced work hours** (either too low or inconsistent).

## 📂 Dataset Description

The dataset includes:

- Satisfaction Level  
- Last Evaluation  
- Number of Projects  
- Average Monthly Hours  
- Time in Company  
- Work Accident  
- Left (0/1)  
- Promotion in Last 5 Years  
- Department  
- Salary Level  

A separate column **Status** was created for visualization:
- 0 → Stayed  
- 1 → Left

This project provides a clear understanding of **why employees leave the company**.
